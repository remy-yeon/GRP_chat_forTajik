# 🚀 Mac 환경에서 Ollama + Gemma2:2b 설치 가이드

이 문서는 **macOS에서 Ollama를 설치하고 Gemma2:2b 모델을 다운로드하는
방법**을 정리한 GitHub용 README입니다.

------------------------------------------------------------------------

## 📦 1. Ollama 설치

Homebrew 또는 공식 스크립트를 사용해 설치할 수 있습니다.\
둘 중 **하나만 선택하여 실행**하면 됩니다.

### **방법 1 --- Homebrew 사용**

``` bash
brew install ollama
ollama serve
```

### **방법 2 --- 공식 설치 스크립트**

``` bash
curl -fsSL https://ollama.com/install.sh | sh
```

------------------------------------------------------------------------

## 🤖 2. Gemma2:2b 모델 다운로드 및 실행

### **실행하면서 자동 다운로드**

``` bash
ollama run gemma2:2b
```

### **다운로드만**

``` bash
ollama pull gemma2:2b
```
